# Matplotlib Visualization Notebook

This repository contains my practice and experiments with **Matplotlib** using Jupyter Notebook.
The goal of this project is to understand how to visualize data effectively using Python.

## About Matplotlib

Matplotlib is one of the most popular Python libraries for creating static, animated, and interactive visualizations.
It is widely used in data science, machine learning, and scientific computing.

## Topics Covered

* Introduction to Matplotlib
* Line plots
* Bar charts
* Scatter plots
* Histograms
* Labels and titles
* Grid and styling
* Multiple plots in a figure

## Tools Used

* Python
* Jupyter Notebook
* NumPy
* Matplotlib
